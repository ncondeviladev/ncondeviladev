### 🚀 Proyecto Destacado: FixFinder *(TFG DAM 2026)*

Sistema distribuido de gestión de incidencias y mantenimiento, diseñado y construido desde cero.

- **Arquitectura:** Ecosistema de tres capas — App Móvil (Flutter), Dashboard Desktop (JavaFX)
  y Servidor Central (Java puro).
- **Desafío Técnico:** Implementación manual de **Sockets TCP** con protocolo binario propio,
  gestión de concurrencia con semáforos y ThreadLocal, sin frameworks de alto nivel.
- **Infraestructura:** Desplegado en producción real sobre **AWS EC2 + RDS**, con
  almacenamiento multimedia en **Firebase Storage**.
- **Stack:** Java 21 · Flutter/Dart · MySQL · Firebase · AWS · Docker

🔗 [Ver repositorio](https://github.com/ncondeviladev/FixFinder)
